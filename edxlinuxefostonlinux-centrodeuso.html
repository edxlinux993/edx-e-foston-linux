Central de Uso - EdxLinux e FostonLinux

Para baixar os pacotes .deb, clique no link abaixo:

Baixar Pacotes .deb

Use o comando abaixo para baixar todos os pacotes .deb com o wget:

wget -r -np -nH --cut-dirs=1 -R "index.html*" https://github.com/edxlinux993/edx-e-foston-linux

Depois de baixar, execute o seguinte comando para instalar todos os pacotes:

sudo dpkg -i *.deb
